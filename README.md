NextPG – PG Finder System

Description

NextPG is a MERN stack based web application that helps users find Paying Guest (PG) accommodations easily.

Features

- Search PG by City
- Filter PG by Rent
- Filter PG by Food Facility
- Add PG
- Edit PG
- Delete PG

Technologies Used

- MongoDB
- Express.js
- React.js
- Node.js
